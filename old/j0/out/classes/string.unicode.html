getLengthAt

(
  • str
  • index
)
Number

Defined in string/unicode.js:10

Unicode is a bitch. While JS uses utf16 characters, it is still possible to have encoded values of multiple UTF-16 characters to represent a user character. This function tries to unravel this mystery. This is somewhat similar to ES6 codePointAt. See the polyfill here: https://developer.mozilla.org/cs/docs/Web/JavaScript/Reference/Global_Objects/String/codePointAt It does a poor job though. There are things like variant-selectors which take multiple code points. For any decent UX app, these need to be considered together as one character.
